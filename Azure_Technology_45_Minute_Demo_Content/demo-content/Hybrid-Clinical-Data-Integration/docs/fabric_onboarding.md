# Microsoft Fabric onboarding (guide)
1. Ensure Fabric is enabled for your tenant (Microsoft 365 admin settings).
2. Create a Fabric capacity (F SKU) or use trial (as permitted) in **US region**.
3. Create a Workspace for Clinical Analytics; assign users/groups via Entra ID.
4. Create a Lakehouse; configure ADF pipelines to land curated data (Delta).
5. Build Power BI semantic models and reports; enforce row-level security via AAD groups.
